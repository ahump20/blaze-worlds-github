Authentication

All API requests require authentication using API keys. Include your API key in the Authorization header:

Authentication Header 
Authorization: Bearer YOUR_API_KEY
Getting Your API Key
  1. Sign up at portal.blaze-intelligence.com
  2. Navigate to Settings → API Keys
  3. Generate a new key for your application
  4. Store securely (keys cannot be recovered)

Vision AI Analysis

Analyze video content for biomechanics, micro-expressions, and character assessment with championship-level precision.

POST /api/v1/vision/analyze
Parameter Type Required Description
video_url string Required URL to video file (MP4, MOV, AVI)
analysis_types array Required Types of analysis: biomechanics, micro_expressions, character
sport string Optional Sport type for context
position string Optional Player position for targeted analysis
Example Request 
const response = await fetch('https://api.blaze-intelligence.com/v1/vision/analyze', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer YOUR_API_KEY',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    video_url: 'https://example.com/quarterback-practice.mp4',
    analysis_types: ['biomechanics', 'micro_expressions', 'character'],
    sport: 'football',
    position: 'quarterback'
  })
});

const analysis = await response.json();
console.log('Championship Readiness:', analysis.championship_metrics.readiness_score);

ROI Calculator

POST /api/v1/roi/calculate
Calculate Program ROI 
const roiResult = await fetch('https://api.blaze-intelligence.com/v1/roi/calculate', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer YOUR_API_KEY',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    program_type: 'college',
    team_size: 85,
    games_per_season: 12,
    current_budget: 8500000,
    sport: 'football',
    goals: ['championship', 'recruiting']
  })
});

const roi = await roiResult.json();
console.log(`ROI: ${roi.roi_analysis.roi_percentage}%`);
console.log(`Payback: ${roi.roi_analysis.payback_period_months} months`);

NIL Analytics

GET /api/v1/nil/athletes/{athlete_id}/valuation
Get Athlete NIL Valuation 
const valuation = await fetch('https://api.blaze-intelligence.com/v1/nil/athletes/tx_qb_001/valuation', {
  headers: {
    'Authorization': 'Bearer YOUR_API_KEY'
  }
});

const nilData = await valuation.json();
console.log(`Estimated NIL Value: $${nilData.valuation.estimated_value.toLocaleString()}`);
console.log(`Market Rank: #${nilData.valuation.market_rank}`);
